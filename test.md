# ARAŞTIRMA PROJESİ
## Sunucu Sistemleri, Terminal Kullanımı ve SSH

### Amaç
Bu proje ile sunucu sistemleri, terminal kullanımı ve SSH teknolojisi hakkında temel bilgi edinmeniz amaçlanmaktadır. Araştırmalarınızı kendi cümlelerinizle hazırlayınız.

---

## 1. Sunucu (Server) Nedir?

Aşağıdaki soruları araştırınız ve cevaplayınız:

* Server (Sunucu) nedir?
* Client (İstemci) nedir?
* Server ile Client arasındaki fark nedir?
* Günlük hayatta kullandığınız en az 5 adet server örneği yazınız.
* Bir web sitesi açıldığında sunucu nasıl çalışır?

## 2. İşletim Sistemleri

Aşağıdaki işletim sistemlerini araştırınız:

* Windows Server
* Ubuntu Server
* Debian
* Rocky Linux (veya CentOS)

**Her işletim sistemi için aşağıdaki soruları cevaplayınız:**

* Kim tarafından geliştirilmiştir?
* Nerelerde kullanılır?
* Avantajları nelerdir?
* Dezavantajları nelerdir?

## 3. Terminal (Komut Satırı)

Aşağıdaki konuları araştırınız:

* Terminal nedir?
* CMD nedir?
* PowerShell nedir?
* Linux Terminal nedir?
* Yazılımcılar neden terminal kullanır?
* Grafik arayüz (GUI) ile Terminal arasındaki farklar nelerdir?

## 4. SSH

Aşağıdaki soruları araştırınız:

* SSH nedir?
* SSH ne işe yarar?
* SSH nasıl çalışır?
* SSH hangi portu kullanır?
* Telnet ile SSH arasındaki fark nedir?
* SSH neden güvenli kabul edilir?
